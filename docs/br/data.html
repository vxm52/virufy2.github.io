Dados abertos

Decidimos tornar públicos alguns de nossos conjuntos de dados de tosse COVID-19.

Queremos beneficiar a humanidade e decidimos ser pioneiros no fornecimento de código aberto do primeiro conjunto de dados de tosse COVID-19 gratuito.

A pandemia de COVID-19 é extremamente prejudicial para a sociedade e todos nós precisamos trabalhar juntos para superá-la. A inteligência artificial precisa de uma grande quantidade de dados para funcionar e não conhecemos em nenhum lugar do mundo um banco de dados de tosse do COVID-19 que seja público. Queremos beneficiar a humanidade e decidimos ser os pioneiros em tornar público o primeiro banco de bados gratuito de tosse do COVID-19. Isso estimulará a inovação e a colaboração de pesquisas acadêmicas e da indústria em todo o mundo.

Fonte de dados

Nossos dados são de altíssima qualidade porque foram coletados em um hospital sob supervisão de médicos de acordo com os procedimentos operacionais padrão (POP). Nossos dados são pré-processados ​​e rotulados com o status COVID-19 (adquiridos a partir de testes de PCR), juntamente com os dados demográficos dos pacientes (idade, sexo, histórico médico).

Iniciar

Nossos dados e instruções de uso podem ser encontrados em nossa página do GitHub. Nossos modelos textuais e de tosse para processar os dados também estão disponíveis.

Colaboração

Revisões de nossos dados e códigos são bem-vindas. Esperamos criar uma comunidade de pesquisadores motivada a usar esses dados para criar soluções para a pandemia. Envie um email para open-data@virufy.org se você estiver interessado.
